07 — Callouts / admonitions
Note
The toLLMContext function returns a Markdown string by default. Pass { format: "plain" } to get plain text instead.
Warning
Quoted reply detection is heuristic-based. Accuracy varies across non-standard email clients.
Tip
Strip URLs before passing to your LLM. They add token count with no semantic value.
